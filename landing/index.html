Возможности

Всё для роста — в одном месте

Древо роста

Пентагон из 5 осей жизни. XP за задачи двигает вершины. Затухает за 30 дней без действий — честный портрет тебя сейчас.

ИИ-ассистент

Диалоговый онбординг, генерация задач по твоему уровню, еженедельная рефлексия. Нейронка помнит контекст.

Задачи + рефлексия

Привязка к осям, XP за выполнение. После завершения — мини-опрос (легко/нормально/сложно), который корректирует XP и учит ИИ.

Помодоро

Встроенный таймер с фазами фокуса и отдыха. Старт прямо из задачи — один тап, и ты в потоке.

Календарь

Месячный вид с точками активности. Клик по дню — завершённые задачи, дедлайны, заметки. Хитмэп активности за год.

База знаний

Персональный граф, который ИИ использует для генерации задач и понимания твоего уровня. Растёт автоматически.

Архитектура

Три слоя на одной оси времени

01

Сейчас

Дашборд с текущей задачей, пульсом недели и помодоро. Фокус на одном действии прямо сейчас.

02

Я

Древо роста — пентагон осей с XP, уровнями и эпохами. Каждая задача двигает вершины. Видишь, кем становишься.

03

Прошлое

Журнал-мемуар с явными промежутками. Время между записями — тоже часть истории.

Философия

Не приложение для продуктивности.
Инструмент для тех, кто хочет расти.

Честность, не мотивация

XP затухает за 30 дней. Пентагон показывает тебя сейчас, а не сумму прошлых побед.

Рефлексия, не галочки

Задача — не чекбокс. После выполнения записываешь как прошло, что мешало. ИИ учится.

Приватно и локально

SQLite на устройстве. Облачная синхронизация опциональна. Твои мысли — только твои.

Минимализм

Строгий ч/б дизайн. Никаких градиентов и ярких цветов. Фокус на содержании.
